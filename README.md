# ZRL-Project-Management-System
ZRL项目管理平台是一个基于ASP.NET Core和达梦数据库构建的企业级项目协同管理系统。系统采用前后端分离架构，前端使用jQuery EasyUI框架提供直观的操作界面，后端通过RESTful API提供稳定高效的数据服务。平台集成了项目管理、成员智能分组、数据可视化展示和配置热更新等核心功能，特别实现了根据成员职位自动匹配工作组的智能机制，支持Excel数据批量导入导出，并确保在不重启服务的情况下动态更新系统配置。该系统专为需要与国产数据库深度集成的企业环境设计，提供了安全可靠的项目协同管理解决方案，适用于各类中小型企业的项目管理需求。
